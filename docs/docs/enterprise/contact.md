# Get in touch

Bancony works with banks and fintechs on production deployments of the [Enterprise MCP server](./mcp-server), the [SDK](./sdk), the [Chat Agent](./chat-agent), and the [Advisory Agents](./advisory-agents).

## How to reach us

- **Web** — [bancony.com](https://bancony.com)
- **Email** — `hello@bancony.com`
- **GitHub** — [@bank2ai](https://github.com/bank2ai) for issues against the open standard

## What to include

If you're evaluating Bancony for a deployment, the most useful first message contains:

- **Your bank/org and market(s).**
- **Which channels you'd integrate with** — mobile app, web banking, branch, internal staff tools.
- **What you've tried so far** — open `bank2ai`, an in-house prototype, nothing yet.
- **Your timeline** — pilot in 3 months? Production in 12?
- **Any specific products** above you're most interested in.

We respond within a few business days.

## Open source first

If you want to evaluate bank2ai itself rather than commercial products, you don't need to talk to us — start with the [Quickstart](/docs/getting-started/quickstart). The open spec, the Python library, and the demo server are all you need to build a working server. Reach out only when you want enterprise support.
